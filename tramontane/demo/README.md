# TRAMONTANE — Mistral Router Demo

**Mistral-native agent orchestration.**
Route intelligently. Govern completely. Ship on EU infrastructure.

This demo shows Tramontane's core feature: the **intelligent model router**
that automatically selects the right Mistral model for your task —
saving cost without sacrificing quality.

## What This Shows

Enter any task description and see:
- Which Mistral model the router selects (and why)
- Estimated cost in EUR
- Full routing decision breakdown
- The complete Mistral fleet with pricing

```bash
pip install tramontane
```
